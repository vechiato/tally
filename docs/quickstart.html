Quick Start

Get Tally running in 5 minutes

1 Install Tally

Choose your platform:

# Linux / macOS
curl -fsSL https://tallyai.money/install.sh | bash

# Windows PowerShell
irm https://tallyai.money/install.ps1 | iex

2 Create a Budget Folder

Initialize a new folder to hold your configuration and data:

tally init ./my-budget
cd my-budget

This creates:

  • settings.yaml - Configuration file
  • config/merchants.rules - Transaction categorization rules
  • data/ - Folder for your CSV files
  • AGENTS.md - Instructions for AI assistants

3 Add Your Bank CSVs

Download transaction exports from your bank and credit cards, then copy them to the data/ folder.

4 Configure Data Sources

Edit settings.yaml to point to your CSV files:

year: 2025

data_sources:
  - name: Chase
    file: data/chase.csv
    format: "{date:%m/%d/%Y},{description},{amount}"

  - name: Bank of America
    file: data/bofa.csv
    format: "{date:%m/%d/%Y},{description},{amount}"

Use tally inspect data/yourfile.csv to see the CSV structure and get format suggestions.

5 Run the Workflow

Check what to do next:

tally workflow

This command detects your setup state and shows context-aware instructions. If you have an AI assistant (Claude Code, Copilot, Cursor), it will guide it through categorizing your transactions.

6 Generate Your Report

Once transactions are categorized:

tally run

This generates an HTML spending report with all your transactions organized by category, recurring expenses identified, and monthly trends visualized.
